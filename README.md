## Installation :

```sh
$ npm install w8.js
```
## Using :

```sh
wait(6000).then().then()...
```
## Example :
```sh
var wait  = require('w8.js');

wait(4000).then(function(){
	console.log("Hello world");
});
// -> will print "Hello World" After 4 Seconds.
```
